# 论文写作规范

## 第一部分 标点格式规范

### 引号铁律

必须用中文弯引号 “ ” U+201C 和 U+201D。

禁止使用英文直引号 “ U+0022。

### 核查命令

```
grep -n '"' filename.md
grep -n '\*\*' filename.md
```

### AI 工具修复

create_file 工具写入文件时，中文弯引号会被转义成英文直引号。文件创建后必须用 Python 脚本批量替换：

```python
with open(filepath, 'r', encoding='utf-8') as f:
    text = f.read()

result = []
in_quote = False
for ch in text:
    if ch == '"':
        if not in_quote:
            result.append('\u201c')
            in_quote = True
        else:
            result.append('\u201d')
            in_quote = False
    else:
        result.append(ch)

with open(filepath, 'w', encoding='utf-8') as f:
    f.write(''.join(result))
```

验证：grep -n '“' filename.md，输出为空即合格。

### 标点规范

所有标点用中文全角，包括，。：；？！” “。

不用省略号 ...，用中文省略号 ……。

插入语一般不允许用，极端情况下一篇文章允许用 1 至 2 次。

破折号一般不允许用，极端情况下一篇文章允许用 1 至 2 次。地名连接号如实皆—曼德勒、并列术语如强制—认知—汲取—渗透保留。

括号尽量不用，但不等于完全不用。中文还原外语原文、必要的限定说明等场景仍可保留。

### 引号使用规则

尽量不用引号。

专有名词、概念术语、专门译法、原文摘引等情况下可以用引号。

必须用中文弯引号 ” “，禁止英文直引号。

### 代词规范

不用这一、那一等指代不明的代词。

不用他、她、它指代不明确的主语。

### 格式清理（提交前必查）

```
grep -n '"' 文件名.md
grep -n '\*\*' 文件名.md
```

英文直引号 0，Markdown 加粗 0。括号数量检查，能删则删。

## 第二部分 AI 痕迹规避

AI 写作的核心病灶在于：用代词、复述、抽象判断、修饰堆砌制造表面严谨。人类写作不会想到那么多代词，也不会复述对方已经讲过的内容。

以下九类表述一律规避，每类附正误对照：

一、假性对立句型。不是…而是…、不仅…更…。形式上提供对比，实际上不提供新信息。

❌ 该文不是简单的描述性研究，而是具有理论建构的雄心。
✅ 该文在梳理若开冲突经过的基础上，提出了"合法性竞争"的分析框架。

二、横向对比赞誉。在硕士论文中实属难得、属于上乘、明显高于同类。只需评价当前对象本身。

❌ 该文的理论意识在同类研究中实属难得，明显优于现有文献。
✅ 该文对2017年若开危机中缅甸军队行动逻辑的分析层次清晰。

三、数字代词堆砌。三个矛盾在三个时期内三重爆发。同一概念在同一段反复用三组、三重、三者，是 AI 语感而非人类语感。

❌ 三个行为体在三个阶段内呈现三重博弈结构。
✅ 中国、缅甸军政府与若开军在2021年政变后各自形成独立的谈判逻辑，三方之间并不存在统一的议价框架。

四、空洞赞誉。体现了良好的学术自觉、展现了扎实的研究能力。不附具体证据等于没有评价。

❌ 作者展现了扎实的学术积累和良好的问题意识。
✅ 作者对1962至2011年军政府执政时期的缅文档案做了系统梳理，引用材料覆盖内比都国家档案馆馆藏。

五、文言化过渡词。彼时、亦有、更为难得的是、有鉴于此、愈益、寻觅、根深蒂固、有所裨益。

❌ 彼时缅北局势愈益复杂，亦有学者指出其内在矛盾，有鉴于此，本文拟从合法性角度切入。
✅ 2023年底缅北局势快速恶化，多位研究者指出军政府对果敢地区的管控已基本失效，本文从合法性角度切入。

六、冗余尾词。句子末尾挂抽象名词（训练、呈现、表现、效果）把前面的动作包起来，显得完整，实际上在重复已说过的意思。处理方式：砍掉末尾包装词，让动词直接收尾。

❌ 论文对东盟在缅甸问题上的立场得到了较为全面的呈现。
✅ 论文梳理了2021至2023年东盟各国在缅甸问题上的分歧，尤以印尼与泰国的立场差异为重点。

❌ 利用Python或R语言完成从数据采集到分析呈现的全流程训练。
✅ 利用Python或R语言独立完成数据采集、清洗、分析与可视化。

七、列举式框架描述。把作者的章节结构、要素清单原样复述。作者按四要素展开，评语里不需要把四个词再列一遍，写每一个时期都有清晰的分析框架即可。

❌ 论文分四章展开，分别分析政治合法性、族群关系、军民矛盾和外部干预，结构完整。
✅ 论文把军政府的合法性危机拆解为国内政治与国际承认两条线索，追溯路径清楚。

八、复述文献综述。把作者的分类与评述再写一遍。AI 倾向于通过复述显得自己读懂了，人类直接给出判断。

❌ 作者将现有研究分为历史制度主义、理性选择和建构主义三类，逐一评述各类文献的贡献与不足。
✅ 现有研究对缅甸军队政治文化的解释停留在威权韧性框架，缺乏对士兵个体动机的分析，该文填补了这一空缺。

九、过度使用指代词。这一、上述、该、此种、这种。AI 用代词连缀句子，人类直接重复关键名词或换用具体表述。

❌ 若开军于2023年10月发动攻势，此举打破了此前的停火安排，这一行动引发了各方的强烈反应。
✅ 若开军于2023年10月发动攻势，终止了2021年以来的停火安排，缅甸军政府随即宣布全面反攻。

### 替代策略

用作者原文表述代替二次概括。

用具体证据替代抽象判断。

用动词主语替代修饰堆砌。

用关键名词重复替代代词指代。

### 自查方法

完成稿件后逐条对照九类清单。

重点检查：同一数字代词是否反复出现；是否出现不是…而是…句型；是否出现彼时、亦有、更为难得的是；是否复述了作者的分类。
